Core Features

Three
Actions

Live

See

Live preview panel — no more alt-tabbing

A persistent side panel in Chrome that shows your dev server output in real-time. Auto-detects your local dev server and refreshes instantly when files change.

  • Auto-detects React, Vue, Svelte, plain HTML
  • Hot-reload aware instant updates
  • Mobile / tablet / desktop viewport controls
  • Visual diff overlay for change tracking
Smart

Pin

Click-to-annotate with DOM-anchored pins

Click any element on the page, drop a pin, type a note. Annotations are anchored to DOM elements — they survive reloads, resizes, and layout shifts.

  • DOM-anchored CSS selectors, not pixel coords
  • Rich context: source file, styles, screenshot
  • Export as JSON, Markdown, or AI prompt
  • Cmd+Shift+A toggles annotation mode
Phase 2

Fix

Direct CSS manipulation with source write-back

Click an element, adjust CSS visually — colors, spacing, typography — and watch changes write back to your source files through AST-aware code modification.

  • Color pickers, spacing sliders, font selectors
  • Writes to CSS, JSX, Tailwind, Vue SFC
  • AST-aware: never corrupts code
  • Instant preview before committing

Architecture

Local.
Secure.
Fast.

Two components connected by WebSocket. The bridge watches your files and resolves source maps. The extension provides the visual layer. Everything stays on your machine.

Preact Fastify WebSocket Chokidar PostCSS Babel AST Vite TypeScript
